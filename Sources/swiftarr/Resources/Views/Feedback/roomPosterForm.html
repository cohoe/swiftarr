Shadow Event Room Names

This page is used to print out a set of Event Feedback Request sheets, to be posted in rooms aboard the ship that are used to host Shadow Events. Shadow Event hosts can then scan the QR code on the sheet and provide feedback on how their event went.
Each sheet printed has the name of the room on it, and the QR code on each sheet is specific to that room. That is, the QR Code's URL has the form: https://twitarr.com/eventfeedback?room=Rolling%20Stone%20Lounge
All this URL does is present the user a filtered list of events; only events happening in the given room are shown. They can still view a full list of all shadow events if their event doesn't appear in the filtered list.
  • Notes on This Form
  • You may want to remove any locations from the list where it won't make sense to post a feedback form. For example, the Caribbean cruises often have shadow events in Half Moon Cay, but THO is unlikely to post a sign there.
  • Be careful about modifying room names on this form. The names are pulled from Sched.com's output, and get used to filter a search of the Event table. Simply changing "Lower Main Dining Room" to "Main Dining" may cause the room filter to not produce any results.
  • If you want to (re)print just one page, you can just delete every line except the one you want and then tap Print.
  • If you are printing on the boat, you may have to Print->Save to PDF, switch Wi-Fi networks to the one the printer's on, and then print the PDF file. This is because your browser may need to fetch images from the server while printing, but your computer (probably) can't see the server and the printer at the same time.
